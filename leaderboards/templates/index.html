Leaderboard explorer

Data Description

The default leaderboard on the left (Uniform_sample_500) corresponds to model rankings on a randomly sampled set of 500 Wildchat/LMArena instances. The default leaderboards on the right (DWA leaderboards) corresponds to the individual leaderboards for each DWA. You can alternatively select Aggregated_DWA_data to see the leaderboard when all the DWA data is aggregated into a single benchmark.

Usage Instructions

Click on rows to track model performance across leaderboards. Hover over the mini-leaderboards to view the full title. Click on the mini-leaderboards to expand and view prompts & model responses (model responses will be displayed for the tracked models).
